# Movie Recommender System

A content-based movie recommendation system built using machine learning.

## Features

Recommends similar movies
Uses cosine similarity
Fetches posters from TMDB API
Interactive UI using Streamlit

## Dataset
TMDB movie dataset

## Tech Stack
Python
Pandas
Scikit-learn
Streamlit

## Run locally

pip install -r requirements.txt

streamlit run app.py

## Example

Input: Batman

Output:
Batman Begins
The Dark Knight
Man of Steel
Superman Returns
Justice League
